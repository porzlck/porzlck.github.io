P2155 [SDOI2008] 沙拉公主的困惑

题目描述

大富翁国因为通货膨胀，以及假钞泛滥，政府决定推出一项新的政策：现有钞票编号范围为 111 到 NNN 的阶乘，但是，政府只发行编号与 M!M!M! 互质的钞票。房地产第一大户沙拉公主决定预测一下大富翁国现在所有真钞票的数量。现在，请你帮助沙拉公主解决这个问题，由于数量可能非常大，你只需计算出答案对 RRR 取模后的结果即可。

输入格式

第一行为两个整数 TTT 和 RRR，其中 TTT 为该组中测试数据数目，RRR 为模数。

接下来 TTT 行，每行一对整数 NNN 和 MMM，具体意义见题目描述。

输出格式

共 TTT 行，对于每一对 NNN 和 MMM，输出 [1,N!][1, N!][1,N!] 中与 M!M!M! 互质的数的数量对 RRR 取模后的值。

输入输出样例 #1

输入 #1

输出 #1

说明/提示

对于 100%100\%100% 的数据，1≤M≤N≤1071\leq M\leq N\leq 10^71≤M≤N≤107，1≤T≤1041\leq T\leq 10^41≤T≤104，2≤R≤109+102\leq R\leq 10^9+102≤R≤109+10 且 RRR 为质数。

题解

设答案为 ansansans，表示区间 [1,N!][1,N!][1,N!] 中与 M!M!M! 互质的整数个数。

一、先推出公式

因为 M≤NM \le NM≤N，所以 M!M!M! 一定整除 N!N!N!。

也就是说，区间 [1,N!][1,N!][1,N!] 可以被分成 N!M!\dfrac{N!}{M!}M!N!​ 个长度恰好为 M!M!M! 的完整块。

而在每一个长度为 M!M!M! 的块里，与 M!M!M! 互质的数的个数都等于 φ(M!)\varphi(M!)φ(M!)。

所以：

ans=N!M!φ(M!)ans=\frac{N!}{M!}\varphi(M!) ans=M!N!​φ(M!)

接下来用欧拉函数公式：

φ(M!)=M!∏p≤M, p为质数p−1p\varphi(M!)=M!\prod_{p\le M,\ p\text{为质数}}\frac{p-1}{p} φ(M!)=M!p≤M, p为质数∏​pp−1​

代回去得到：

ans=N!∏p≤M, p为质数p−1pans=N!\prod_{p\le M,\ p\text{为质数}}\frac{p-1}{p} ans=N!p≤M, p为质数∏​pp−1​

二、为什么要单独讨论 R

题目要求答案对 RRR 取模，并且 RRR 是质数。

上面的式子里有很多分母 ppp。

  • 如果 p≠Rp\ne Rp​=R，那么因为 RRR 是质数，所以 gcd⁡(p,R)=1\gcd(p,R)=1gcd(p,R)=1，于是 ppp 在模 RRR 意义下一定有逆元，可以正常计算。
  • 只有当 p=Rp=Rp=R 时，分母里的这个 RRR 在模 RRR 下没有逆元，所以必须单独讨论。

并且注意，在这个乘积里，每个质数只出现一次，所以最多只会出现一个 RRR。

三、分情况讨论

1. N < R

这时 N!N!N! 不含因子 RRR，所以不会出现模 RRR 下直接变成 000 的问题。

又因为题目保证 M≤NM\le NM≤N，所以既然 N<RN<RN<R，就一定有 M<RM<RM<R。

于是乘积里也不会出现 p=Rp=Rp=R 这一项。

因此这时可以直接计算：

ans=N!∏p≤M, p为质数p−1p(modR)ans=N!\prod_{p\le M,\ p\text{为质数}}\frac{p-1}{p}\pmod R ans=N!p≤M, p为质数∏​pp−1​(modR)

2. R ≤ N < 2R

这时 N!N!N! 中恰好只有一个因子 RRR。

因为在 1,2,⋯ ,N1,2,\cdots,N1,2,⋯,N 中：

  • 有一个数是 RRR
  • 但还没有到 2R2R2R，所以没有第二个 RRR 的倍数

于是分两种情况。

(1) M < R

这时乘积里没有 p=Rp=Rp=R 这一项，没法把 N!N!N! 里的那个 RRR 消掉。

所以答案一定是：

ans=0ans=0 ans=0

(2) M ≥ R

这时乘积里恰好会出现一项：

R−1R\frac{R-1}{R} RR−1​

它可以把 N!N!N! 里唯一的那个 RRR 消掉。

于是：

ans=N!R(R−1)∏p≤M, p≠R, p为质数p−1p(modR)ans=\frac{N!}{R}(R-1)\prod_{p\le M,\ p\ne R,\ p\text{为质数}}\frac{p-1}{p}\pmod R ans=RN!​(R−1)p≤M, p​=R, p为质数∏​pp−1​(modR)

下面化简 N!R\dfrac{N!}{R}RN!​。

因为：

N!=1⋅2⋅⋯⋅(R−1)⋅R⋅(R+1)⋅⋯⋅NN!=1\cdot2\cdot\cdots\cdot(R-1)\cdot R\cdot(R+1)\cdot\cdots\cdot N N!=1⋅2⋅⋯⋅(R−1)⋅R⋅(R+1)⋅⋯⋅N

所以：

N!R=(R−1)!⋅(R+1)(R+2)⋯N\frac{N!}{R}=(R-1)!\cdot(R+1)(R+2)\cdots N RN!​=(R−1)!⋅(R+1)(R+2)⋯N

在模 RRR 下：

R+1≡1,R+2≡2,⋯ ,N≡N−RR+1\equiv1,\quad R+2\equiv2,\quad \cdots,\quad N\equiv N-R R+1≡1,R+2≡2,⋯,N≡N−R

所以：

(R+1)(R+2)⋯N≡1⋅2⋅⋯⋅(N−R)=(N−R)!(modR)(R+1)(R+2)\cdots N\equiv 1\cdot2\cdot\cdots\cdot(N-R)=(N-R)!\pmod R (R+1)(R+2)⋯N≡1⋅2⋅⋯⋅(N−R)=(N−R)!(modR)

于是：

N!R≡(R−1)!(N−R)!(modR)\frac{N!}{R}\equiv (R-1)!(N-R)!\pmod R RN!​≡(R−1)!(N−R)!(modR)

再由 Wilson 定理：

(R−1)!≡−1(modR)(R-1)!\equiv -1\pmod R (R−1)!≡−1(modR)

并且：

R−1≡−1(modR)R-1\equiv -1\pmod R R−1≡−1(modR)

所以这两个乘起来刚好等于 111，最后得到：

ans=(N−R)!∏p≤M, p≠R, p为质数p−1p(modR)ans=(N-R)!\prod_{p\le M,\ p\ne R,\ p\text{为质数}}\frac{p-1}{p}\pmod R ans=(N−R)!p≤M, p​=R, p为质数∏​pp−1​(modR)

3. N ≥ 2R

这时 N!N!N! 中至少有两个因子 RRR。

因为：

  • 一个来自 RRR
  • 另一个至少来自 2R2R2R

而乘积部分至多只能提供一个 1R\dfrac{1}{R}R1​ 去消掉一个 RRR。

所以最后还会剩下一个因子 RRR，答案一定是：

ans=0ans=0 ans=0

四、整理成最终结论

定义：

S(M)=∏p≤M, p≠R, p为质数p−1p(modR)S(M)=\prod_{p\le M,\ p\ne R,\ p\text{为质数}}\frac{p-1}{p}\pmod R S(M)=p≤M, p​=R, p为质数∏​pp−1​(modR)

那么答案就是：

情况 1：N < R

ans=N!⋅S(M)(modR)ans=N!\cdot S(M)\pmod R ans=N!⋅S(M)(modR)

情况 2：R ≤ N < 2R

  • 如果 M<RM<RM<R，那么

ans=0ans=0 ans=0

  • 如果 M≥RM\ge RM≥R，那么

ans=(N−R)!⋅S(M)(modR)ans=(N-R)!\cdot S(M)\pmod R ans=(N−R)!⋅S(M)(modR)

情况 3：N ≥ 2R

ans=0ans=0 ans=0

五、预处理

由于有多组询问，需要预处理。

1. 预处理阶乘

只需要预处理到 min⁡(max⁡N, R−1)\min(\max N,\ R-1)min(maxN, R−1)。

因为真正会用到的只有：

  • N!N!N!，当 N<RN<RN<R
  • (N−R)!(N-R)!(N−R)!，当 R≤N<2RR\le N<2RR≤N<2R，这时 N−R<RN-R<RN−R<R

2. 预处理前缀乘积 S

定义数组 pre[i]pre[i]pre[i]：

pre[i]=∏p≤i, p≠R, p为质数p−1p(modR)pre[i]=\prod_{p\le i,\ p\ne R,\ p\text{为质数}}\frac{p-1}{p}\pmod R pre[i]=p≤i, p​=R, p为质数∏​pp−1​(modR)

那么：

  • 如果 iii 不是质数，pre[i]=pre[i−1]pre[i]=pre[i-1]pre[i]=pre[i−1]
  • 如果 iii 是质数且 i≠Ri\ne Ri​=R，那么

pre[i]=pre[i−1]⋅(i−1)⋅i−1(modR)pre[i]=pre[i-1]\cdot (i-1)\cdot i^{-1}\pmod R pre[i]=pre[i−1]⋅(i−1)⋅i−1(modR)

这里逆元用费马小定理求：

i−1≡iR−2(modR)i^{-1}\equiv i^{R-2}\pmod R i−1≡iR−2(modR)

因为 RRR 是质数。
