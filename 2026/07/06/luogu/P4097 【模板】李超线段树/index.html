P4097 【模板】李超线段树 / [HEOI2013] Segment

发表于2026-07-06|更新于2026-07-06|算法
|浏览量:

题目描述

要求在平面直角坐标系下维护两个操作：

  1. 在平面上加入一条线段。记第 iii 条被插入的线段的标号为 iii。
  2. 给定一个数 kkk，询问与直线 x=kx = kx=k 相交的线段中，交点纵坐标最大的线段的编号。

输入格式

本题输入强制在线。

输入的第一行是一个整数 nnn，代表操作的个数。

接下来 nnn 行，每行若干个用空格隔开的整数，第 i+1i + 1i+1 行的第一个整数为 opopop，代表第 iii 次操作的类型。

若 op=0op = 0op=0，则后跟一个整数 kkk，代表本次操作为查询所有与直线 x=(k+lastans−1) mod 39989+1x = (k + lastans - 1) \bmod 39989 + 1x=(k+lastans−1)mod39989+1 相交的线段中，交点纵坐标最大的线段编号。

若 op=1op = 1op=1，则后跟四个整数 x0,y0,x1,y1x_0, y_0, x_1, y_1x0​,y0​,x1​,y1​，记 xi′=(xi+lastans−1) mod 39989+1x_i' = (x_i + lastans - 1) \bmod 39989 + 1xi′​=(xi​+lastans−1)mod39989+1，yi′=(yi+lastans−1) mod 109+1y_i' = (y_i + lastans - 1) \bmod 10^9 + 1yi′​=(yi​+lastans−1)mod109+1。本次操作为插入一条两端点分别为 (x0′,y0′)(x_0', y_0')(x0′​,y0′​)，(x1′,y1′)(x_1',y_1')(x1′​,y1′​) 的线段。

其中 lastanslastanslastans 为上次询问的答案，初始时，lastans=0lastans = 0lastans=0。

输出格式

对于每次查询，输出一行一个整数，代表交点纵坐标最大的线段的编号。若不存在任何一条线段与查询直线有交，则输出 000；若有多条线段与查询直线的交点纵坐标都是最大的，则输出编号最小的线段，同时 lastanslastanslastans 也应更新为编号最小的一条线段。
