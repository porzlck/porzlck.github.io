CF1748E Yet Another Array Counting Problem

题目描述

对于长度为 nnn 的序列 xxx，定义其在子段 [l;r][l;r][l;r] 的“最左端最大值位置”为最小的满足 l≤i≤rl\leq i\leq rl≤i≤r 且 xi=max⁡j=lrxjx_i=\max_{j=l}^rx_jxi​=maxj=lr​xj​ 的整数 iii。
给定整数 n,mn,mn,m 和长度为 nnn 的序列 aaa，你需要求出满足下列要求的序列 bbb 的数量：

  • 序列 bbb 长度为 nnn，且对任意整数 i(1≤i≤n)i(1\leq i\leq n)i(1≤i≤n) 都有 1≤bi≤m1\leq b_i\leq m1≤bi​≤m 成立。
  • 对任意整数 l,r(1≤l≤r≤n)l,r(1\leq l\leq r\leq n)l,r(1≤l≤r≤n)，总有 a,ba,ba,b 在子段 [l;r][l;r][l;r] 的“最左端最大值位置”相同。

答案对 109+710^9+7109+7 取模。
每个测试点包含多组数据。

输入格式

第一行输入一个整数 t(1≤t≤103)t(1\leq t\leq10^3)t(1≤t≤103) 表示数据组数，接下来对于每组数据：
第一行输入两个整数 n,m(2≤n,m≤2×105;∑n×m≤106)n,m(2\leq n,m\leq2\times10^5;\sum n\times m\leq10^6)n,m(2≤n,m≤2×105;∑n×m≤106)。
接下来输入一行 nnn 个整数表示序列 a(1≤ai≤m)a(1\leq a_i\leq m)a(1≤ai​≤m)。

输出格式

对于每组数据：
输出满足要求的序列 bbb 的数量对 109+710^9+7109+7 取模后的值。

思路

按照“最大值优先、相等时位置更靠左优先”建立大根笛卡尔树。

对树进行值域 DP。

对每个父子关系：

左儿子的值必须严格小于父亲；
右儿子的值可以小于等于父亲。
