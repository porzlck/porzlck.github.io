E.托姆大战汤姆

历经千辛万苦，托姆终于逃离了迷宫。出现在托姆眼前的人，便是梦境世界的制造者——汤姆。只有与汤姆正面交锋并将其击败，托姆才能够彻底逃离梦境世界，并在现实世界中醒来。

场上初始有 nnn 只怪物，第 iii 只怪物的初始血量为 hih_ihi​。

作为战斗的先手，托姆必须首先从这 nnn 只怪物中选出一只，随后汤姆从剩下的 n−1n-1n−1 只怪物中挑出一只，并将其他的怪物移出游戏。

挑选完毕后，托姆和汤姆轮流行动，其中托姆先手。每次行动时，当前玩家必须从场上血量严格大于 000 的怪物中选择一只，对其发起一次攻击。

每次攻击造成的伤害为一个整数，该整数在闭区间 [L,R][L, R][L,R] 内的所有整数中独立且等概率地随机产生 (即离散均匀分布) 。受到攻击的怪物，其血量将直接减去该次造成的伤害值。

当怪物的血量 ≤0\le 0≤0 时，该怪物被判定为死亡，并立即从场上移除，之后不可再被选中。成功击杀场上最后一只存活的怪物的玩家，将获得整场战斗的胜利。

假设双方都极其聪明，在每一次需要做出选择时，都会采取使自己最终获胜概率最大化的最优策略。托姆想知道，如果他在游戏开始时做出最优的怪物选择，最终能够获胜的最大概率是多少？

Input

第一行包含三个正整数 nnn , LLL 和 RRR (2≤n≤2000(2 \le n \le 2000(2≤n≤2000,1≤L≤R≤2000)1 \le L \le R \le 2000)1≤L≤R≤2000) ，分别表示初始场上怪物的总数量，以及每次攻击伤害的下界和上界。

第二行包含 nnn 个正整数 h1,h2,…,hnh_1, h_2, \dots, h_nh1​,h2​,…,hn​ (1≤hi≤2000)(1 \le h_i \le 2000)(1≤hi​≤2000)，表示每只怪物的初始血量 。

Output

输出一个浮点数，表示托姆在双方均采取最优策略的前提下，能够获胜的最大概率。

你的答案将被认为是正确的，当且仅当你的答案与标准答案的绝对误差或相对误差不超过 10−610^{-6}10−6 。

思路

对于一只血量为 xxx 的怪物，设：

odd[x]odd[x] odd[x]

表示这只怪物从当前血量开始，最终需要被攻击奇数次才死亡的概率。

当怪物血量已经为 000 时，不需要再攻击，因此攻击次数为 000，是偶数：

odd[0]=0odd[0]=0 odd[0]=0

假设当前怪物血量为 xxx，本次攻击造成伤害 ddd，攻击后血量为：

nxt=max⁡(0,x−d)nxt=\max(0,x-d) nxt=max(0,x−d)

当前已经进行了一次攻击，因此剩余攻击次数的奇偶性会翻转。

如果剩余还需要偶数次攻击，那么加上当前这一次后，总次数就是奇数。

所以对于伤害 ddd，总攻击次数为奇数的概率是：

1−odd[nxt]1-odd[nxt] 1−odd[nxt]

由于伤害在区间 [L,R][L,R][L,R] 内等概率产生，因此：

odd[x]=1R−L+1∑d=LR(1−odd[max⁡(0,x−d)])odd[x] = \frac{1}{R-L+1} \sum_{d=L}^{R} \left(1-odd[\max(0,x-d)]\right) odd[x]=R−L+11​d=L∑R​(1−odd[max(0,x−d)])

按照血量从小到大计算即可。

两只怪物的获胜概率

假设选择的两只怪物血量分别为 xxx 和 yyy。

托姆先手，双方轮流攻击，因此总攻击次数为奇数时，最后一次攻击由托姆完成，托姆获胜。

两只怪物总攻击次数为奇数有两种情况：

  1. 第一只需要奇数次，第二只需要偶数次；
  2. 第一只需要偶数次，第二只需要奇数次。

因此托姆的获胜概率为：

win(x,y)=odd[x](1−odd[y])+(1−odd[x])odd[y]]win(x,y) = odd[x](1-odd[y]) + (1-odd[x])odd[y] ] win(x,y)=odd[x](1−odd[y])+(1−odd[x])odd[y]]

双方选择怪物

托姆先选择第 iii 只怪物，汤姆再从剩余怪物中选择第 jjj 只。

固定托姆选择 iii 后，汤姆会选择一个让托姆获胜概率最小的 jjj：

worsti=min⁡j≠iwin(hi,hj)]worst_i = \min_{j\ne i} win(h_i,h_j) ] worsti​=j​=imin​win(hi​,hj​)]

也就是：

F.动物园派对

动物园正在举行一场疯狂的动物派对！派对上聚集了来自世界各地的动物，有喜欢美美隐身的鳄鱼 、 躲在地下打洞的老鼠 、 还有到处游走的蜥蜴等 。 它们排成了一排，依次编号为 111 到 nnn 。

为了在嘈杂的派对中保持联系，小动物们需要建立水平激光通讯链路 。 每只小动物 iii 都有一个所在的位置高度 hih_ihi​ （例如长颈鹿的高度是正数，而潜水的鳄鱼高度可能是负数） ， 以及一个通讯频段特征值 aia_iai​ 。

当小动物 lll 和小动物 rrr （ 1≤l<r≤n1 \le l \lt r \le n1≤l<r≤n ） 尝试进行通讯时，为了保证信号不被中间的动物挡住，通讯信号所在的有效高度 HHH 取决于它们之间 （包含它们自己） 所有动物的最低高度，即：

H=min⁡i=lrhiH = \min_{i=l}^r h_i H=i=lminr​hi​

同时，派对专门的环境会对通讯产生干扰，干扰系数为一个常数 KKK 。 链路的通讯强度不仅与有效高度有关，还受到两只动物频段异或值的影响。具体来说，它们之间的通讯强度定义为：

S(l,r)=H×((al⊕ar)−K)S(l, r) = H \times ( ( a_l \oplus a_r ) - K ) S(l,r)=H×((al​⊕ar​)−K)

其中 ⊕\oplus⊕ 表示按位异或运算 。

现在派对到达了高潮，请你作为动物园的网络管理员，计算出在所有可能的通讯对 (l,r)(l, r)(l,r) 中，通讯强度的 最大值 是多少。

Input

第一行包含一个整数 TTT （ 1≤T≤1041 \le T \le 10^41≤T≤104 ） ， 表示测试数据的组数 。

对于每组测试数据： 第一行包含两个整数 nnn 和 KKK （ 2≤n≤2⋅1052 \le n \le 2 \cdot 10^52≤n≤2⋅105 ， 0≤K<2310 \le K \lt 2^{31}0≤K<231 ） ， 分别表示小动物的数量和环境干扰系数 。

第二行包含 nnn 个整数 h1,h2,…,hnh_1, h_2, \dots, h_nh1​,h2​,…,hn​ （ −108≤hi≤108-10^8 \le h_i \le 10^8−108≤hi​≤108 ） ， 表示每只小动物所在的高度 。

第三行包含 nnn 个整数 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​ （ 0≤ai<2310 \le a_i \lt 2^{31}0≤ai​<231 ） ， 表示每只小动物的通讯频段特征值 。

思路

对于任意一对位置 (l<r)，通信强度为：

S(l,r)=min⁡i=lrhi⋅((al⊕ar)−K)S(l,r)=\min_{i=l}^{r}h_i\cdot ((a_l\oplus a_r)-K) S(l,r)=i=lminr​hi​⋅((al​⊕ar​)−K)

难点在于同时处理：

  • 区间最小值；
  • 两个端点的异或值。

1. 建立小根笛卡尔树

对高度数组 hhh 建立小根笛卡尔树，满足：

  • 笛卡尔树的中序遍历顺序为 (1,2,…,n)(1,2,\dots,n)(1,2,…,n)；
  • 父节点的高度不大于子节点的高度。

对于任意两个位置 (l,r)(l,r)(l,r)，设它们在笛卡尔树中的最近公共祖先为 uuu，则有：

min⁡i=lrhi=hu\min_{i=l}^{r}h_i=h_u i=lminr​hi​=hu​

因此原式可以转化为：

S(l,r)=hu⋅((al⊕ar)−K)S(l,r)=h_u\cdot ((a_l\oplus a_r)-K) S(l,r)=hu​⋅((al​⊕ar​)−K)

其中：

u=LCA⁡(l,r)u=\operatorname{LCA}(l,r) u=LCA(l,r)

2. 枚举最近公共祖先

对于笛卡尔树中的一个节点 (u)，所有满足：

LCA⁡(l,r)=u\operatorname{LCA}(l,r)=u LCA(l,r)=u

的点对可以分成三类：

  1. 一个点是 uuu，另一个点在左子树中；
  2. 一个点是 uuu，另一个点在右子树中；
  3. 一个点在左子树中，另一个点在右子树中。

这些点对对应区间的最小高度都等于 huh_uhu​。

3. 使用异或 Trie

对于固定的节点 (u)，需要最大化：

hu⋅((al⊕ar)−K)h_u\cdot ((a_l\oplus a_r)-K) hu​⋅((al​⊕ar​)−K)

当 hu≥0h_u\ge 0hu​≥0 时，乘数为非负数，因此需要让：

al⊕ara_l\oplus a_r al​⊕ar​

尽可能大。

当 hu<0h_u<0hu​<0 时，乘数为负数，因此需要让：

al⊕ara_l\oplus a_r al​⊕ar​

尽可能小。

所以使用二进制 Trie 支持：

  • 查询与一个数的最大异或值；
  • 查询与一个数的最小异或值。

C.无法逃离的梦境

在做出了正确的选择后，托姆渐渐感觉自己的意识离开了虚无之海。然而，当托姆以为自己即将醒来时，他发现他并没有回到现实，而是坠入了更深的一层梦境。这一次，他来到了一个由 hsq 精心设计的迷宫。

这个迷宫中共有 nnn 个房间，编号为 1,2,…,n1, 2, \ldots, n1,2,…,n。每个房间中都有一条无法改变的单向通道：如果托姆当前位于房间 iii，那么在 hsq 对他进行一次操控后，他必然会沿着通道进入房间 pip_ipi​。

除了所在的房间，托姆还拥有一种梦境状态。梦境状态共有 ccc 种，编号为 0,1,…,c−10, 1, \ldots, c-10,1,…,c−1。

在每一次操控时，若托姆当前位于房间 iii，且当前的梦境状态为 aaa，hsq 可以将其梦境状态切换为任意状态 bbb（bbb 可以与 aaa 相同），并为此付出 wi(a,b)w_i(a, b)wi​(a,b) 的代价。这个代价可能为负数，表示这次操控反而让 hsq 获得了某种收益。在状态切换完成之后，托姆会立刻沿着房间 iii 的通道进入房间 pip_ipi​。

现在有 qqq 次询问。每次询问给出四个整数 x,k,s,tx, k, s, tx,k,s,t，表示托姆初始位于房间 xxx，初始梦境状态为 sss。hsq 需要恰好操控托姆 kkk 次，并希望在 kkk 次操控结束后，托姆的梦境状态恰好为 ttt。

请你求出，对于每次询问，hsq 达成目标所需要付出的最小总代价。

第二行包含 nnn 个整数 p1,p2,…,pnp_1,p_2,\ldots,p_np1​,p2​,…,pn​ (1≤pi≤n1 \le p_i \le n1≤pi​≤n)。其中 pip_ipi​ 表示如果托姆当前位于房间 iii，那么 hsq 操控他一次之后，托姆会进入房间 pip_ipi​。

对于每个房间 iii (1≤i≤n1 \le i \le n1≤i≤n)，输入 ccc 行，每行包含 ccc 个整数。

第 a+1a+1a+1 行第 b+1b+1b+1 个整数表示 wi(a,b)w_i(a, b)wi​(a,b) (0≤a,b<c0 \le a,b \lt c0≤a,b<c，−109≤wi(a,b)≤109-10^9 \le w_i(a, b) \le 10^9−109≤wi​(a,b)≤109)，即当托姆位于房间 iii，且当前梦境状态为 aaa 时，hsq 将他的梦境状态切换为 bbb 所需付出的代价。
