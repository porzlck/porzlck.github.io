在区间 [−π,π][-\pi,\pi][−π,π] 上，三角函数系中的任意两个不同函数的乘积积分都等于零。

首先，对于任意正整数 nnn，有

∫−ππcos⁡nx dx=0,\int_{-\pi}^{\pi}\cos nx\,dx=0, ∫−ππ​cosnxdx=0,

∫−ππsin⁡nx dx=0.\int_{-\pi}^{\pi}\sin nx\,dx=0. ∫−ππ​sinnxdx=0.

f(x)=a02+∑n=1∞(ancos⁡nx+bnsin⁡nx).f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n\cos nx+b_n\sin nx \right). f(x)=2a0​​+n=1∑∞​(an​cosnx+bn​sinnx).

此时由定理 15.2 可知，这些系数一定就是函数 fff 的傅里叶系数。

但是，如果只是从一个可积的周期函数 fff 出发，根据积分公式计算出 ana_nan​ 和 bnb_nbn​，并写出对应的三角级数，则暂时只能写成

f(x)∼a02+∑n=1∞(ancos⁡nx+bnsin⁡nx).f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n\cos nx+b_n\sin nx \right). f(x)∼2a0​​+n=1∑∞​(an​cosnx+bn​sinnx).

因为这时还没有证明：

  1. 该傅里叶级数是否收敛；
  2. 若收敛，它是否收敛到 f(x)f(x)f(x)；
  3. 在间断点处收敛到什么值。

这些问题需要进一步研究傅里叶级数的收敛性。

傅里叶级数收敛定理

下面给出傅里叶级数的一个重要收敛定理。

定理 15.3：傅里叶级数收敛定理

设函数 fff 以 2π2\pi2π 为周期，并且在区间 [−π,π][-\pi,\pi][−π,π] 上分段光滑，则对于每一点

x∈[−π,π],x\in[-\pi,\pi], x∈[−π,π],

函数 fff 的傅里叶级数收敛于 fff 在点 xxx 处左、右极限的算术平均值，即

f(x+0)+f(x−0)2=a02+∑n=1∞(ancos⁡nx+bnsin⁡nx),\frac{f(x+0)+f(x-0)}{2} = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n\cos nx+b_n\sin nx \right), 2f(x+0)+f(x−0)​=2a0​​+n=1∑∞​(an​cosnx+bn​sinnx),

其中 an,bna_n,b_nan​,bn​ 是函数 fff 的傅里叶系数。

也就是说，傅里叶级数在点 xxx 处的和为

S(x)=f(x+0)+f(x−0)2.S(x)=\frac{f(x+0)+f(x-0)}{2}. S(x)=2f(x+0)+f(x−0)​.

光滑函数

若函数 fff 的导函数 f′f'f′ 在区间 [a,b][a,b][a,b] 上连续，则称函数 fff 在区间 [a,b][a,b][a,b] 上光滑。

换句话说，如果

f∈C1[a,b],f\in C^1[a,b], f∈C1[a,b],

则 fff 在 [a,b][a,b][a,b] 上光滑。

分段光滑函数

设函数 fff 定义在区间 [a,b][a,b][a,b] 上。如果满足：

  1. fff 在 [a,b][a,b][a,b] 上至多有有限个第一类间断点；
  2. 除了至多有限个不可导点以外，f′f'f′ 连续；
  3. 在这些不可导点处，f′f'f′ 的左、右极限都存在；

则称函数 fff 在区间 [a,b][a,b][a,b] 上分段光滑。

从几何图形上看，分段光滑函数由有限多个光滑弧段组成，因此它至多有有限个跳跃间断点和尖点。

分段光滑函数的性质

若函数 fff 在区间 [a,b][a,b][a,b] 上分段光滑，则有以下性质。

1. 函数可积

函数 fff 在区间 [a,b][a,b][a,b] 上可积。

2. 左、右极限存在

对于区间内每一点 xxx，左右极限

f(x+0),f(x−0)f(x+0),\qquad f(x-0) f(x+0),f(x−0)

都存在。

在端点处只考虑相应的单侧极限。

3. 单侧导数存在

对于每个分段光滑点，有

lim⁡t→0+f(x+t)−f(x+0)t=f′(x+0),\lim_{t\to 0^+} \frac{f(x+t)-f(x+0)}{t} = f'(x+0), t→0+lim​tf(x+t)−f(x+0)​=f′(x+0),

以及

lim⁡t→0+f(x−t)−f(x−0)−t=f′(x−0).\lim_{t\to 0^+} \frac{f(x-t)-f(x-0)}{-t} = f'(x-0). t→0+lim​−tf(x−t)−f(x−0)​=f′(x−0).

这里：

  • f′(x+0)f'(x+0)f′(x+0) 表示右导数；
  • f′(x−0)f'(x-0)f′(x−0) 表示左导数。

4. 导函数可积

在至多有限个导数不存在的点上，补充定义 f′f'f′ 的值后，函数 f′f'f′ 在区间 [a,b][a,b][a,b] 上可积。

连续点处的收敛

若函数 fff 在点 xxx 处连续，则

f(x+0)=f(x−0)=f(x).f(x+0)=f(x-0)=f(x). f(x+0)=f(x−0)=f(x).

因此，收敛定理中的算术平均值变为

f(x+0)+f(x−0)2=f(x).\frac{f(x+0)+f(x-0)}{2} = f(x). 2f(x+0)+f(x−0)​=f(x).

所以在连续点处，函数 fff 的傅里叶级数收敛于函数值 f(x)f(x)f(x)：

f(x)=a02+∑n=1∞(ancos⁡nx+bnsin⁡nx).f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n\cos nx+b_n\sin nx \right). f(x)=2a0​​+n=1∑∞​(an​cosnx+bn​sinnx).

间断点处的收敛

若函数 fff 在点 xxx 处有第一类间断点，则左右极限存在，但通常不相等：

f(x−0)≠f(x+0).f(x-0)\neq f(x+0). f(x−0)​=f(x+0).

此时傅里叶级数不一定收敛到函数在该点的定义值 f(x)f(x)f(x)，而是收敛到左右极限的平均值：

S(x)=f(x−0)+f(x+0)2.S(x) = \frac{f(x-0)+f(x+0)}{2}. S(x)=2f(x−0)+f(x+0)​.

因此，改变函数在有限个间断点处的具体取值，不会改变它的傅里叶系数，也不会改变傅里叶级数。

推论

若函数 fff 是以 2π2\pi2π 为周期的连续函数，并且在区间 [−π,π][-\pi,\pi][−π,π] 上分段光滑，则函数 fff 的傅里叶级数在整个实数轴上收敛于 fff，即

f(x)=a02+∑n=1∞(ancos⁡nx+bnsin⁡nx),x∈R.f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n\cos nx+b_n\sin nx \right), \qquad x\in\mathbb R. f(x)=2a0​​+n=1∑∞​(an​cosnx+bn​sinnx),x∈R.

这里要求函数作为周期函数在周期连接处也是连续的，即应满足

f(−π)=f(π).f(-\pi)=f(\pi). f(−π)=f(π).

若 f(−π)≠f(π)f(-\pi)\neq f(\pi)f(−π)​=f(π)，那么将函数作周期延拓以后，在

x=(2k+1)π,k∈Zx=(2k+1)\pi,\qquad k\in\mathbb Z x=(2k+1)π,k∈Z

处会出现跳跃间断。

在任意长度为 2pi 的区间上计算系数

因为函数 fff 以 2π2\pi2π 为周期，所以傅里叶系数的积分区间不一定必须取为 [−π,π][-\pi,\pi][−π,π]。

对于任意实数 ccc，都可以在长度为 2π2\pi2π 的区间 [c,c+2π][c,c+2\pi][c,c+2π] 上计算：

an=1π∫cc+2πf(x)cos⁡nx dx,n=0,1,2,⋯ ,a_n = \frac{1}{\pi} \int_c^{c+2\pi} f(x)\cos nx\,dx, \qquad n=0,1,2,\cdots, an​=π1​∫cc+2π​f(x)cosnxdx,n=0,1,2,⋯,

以及

bn=1π∫cc+2πf(x)sin⁡nx dx,n=1,2,⋯ .b_n = \frac{1}{\pi} \int_c^{c+2\pi} f(x)\sin nx\,dx, \qquad n=1,2,\cdots. bn​=π1​∫cc+2π​f(x)sinnxdx,n=1,2,⋯.

傅里叶系数的值与积分区间起点 ccc 的选取无关，只要积分区间的长度为一个完整周期 2π2\pi2π 即可。

周期延拓

在具体问题中，常常只给出函数 fff 在区间

(−π,π](-\pi,\pi] (−π,π]

上的解析表达式。

此时应把它理解为一个以 2π2\pi2π 为周期的函数在一个周期内的定义，并将它延拓到整个实数轴。

设原函数在 (−π,π](-\pi,\pi](−π,π] 上定义，则它的周期延拓函数记为 f~\widetilde ff​：

f~(x)={f(x),x∈(−π,π],f(x−2kπ),x∈((2k−1)π,(2k+1)π],k=±1,±2,⋯ .\widetilde f(x) = \begin{cases} f(x), & x\in(-\pi,\pi],\\[6pt] f(x-2k\pi), & x\in((2k-1)\pi,(2k+1)\pi],\quad k=\pm1,\pm2,\cdots. \end{cases} f​(x)=⎩⎨⎧​f(x),f(x−2kπ),​x∈(−π,π],x∈((2k−1)π,(2k+1)π],k=±1,±2,⋯.​

周期延拓后的函数满足

f~(x+2π)=f~(x).\widetilde f(x+2\pi)=\widetilde f(x). f​(x+2π)=f​(x).

因此，通常所说的“函数 fff 的傅里叶级数”，实际上是指它的周期延拓函数 f~\widetilde ff​ 的傅里叶级数。

周期端点处的收敛

在周期延拓以后，点 x=πx=\pix=π 和 x=−πx=-\pix=−π 被视为周期连接处。

因此在端点处，傅里叶级数收敛到

f(−π+0)+f(π−0)2.\frac{f(-\pi+0)+f(\pi-0)}{2}. 2f(−π+0)+f(π−0)​.

更一般地，在所有周期连接点

x=(2k+1)π,k∈Zx=(2k+1)\pi,\qquad k\in\mathbb Z x=(2k+1)π,k∈Z

处，傅里叶级数都收敛到相邻两个周期的左右极限平均值。

如果满足

f(−π+0)=f(π−0),f(-\pi+0)=f(\pi-0), f(−π+0)=f(π−0),

那么周期延拓函数在连接处连续，傅里叶级数在这些点收敛到共同的函数值。

如果

f(−π+0)≠f(π−0),f(-\pi+0)\neq f(\pi-0), f(−π+0)​=f(π−0),

那么周期延拓函数在连接处发生跳跃，傅里叶级数在这些点收敛到

f(−π+0)+f(π−0)2.\frac{f(-\pi+0)+f(\pi-0)}{2}. 2f(−π+0)+f(π−0)​.

收敛定理总结

对于以 2π2\pi2π 为周期、并在一个周期内分段光滑的函数 fff，其傅里叶级数满足

S(x)=f(x−0)+f(x+0)2.S(x) = \frac{f(x-0)+f(x+0)}{2}. S(x)=2f(x−0)+f(x+0)​.

因此：

  • 若 fff 在 xxx 处连续，则

    S(x)=f(x).S(x)=f(x). S(x)=f(x).

  • 若 fff 在 xxx 处发生跳跃，则

    S(x)=f(x−0)+f(x+0)2.S(x) = \frac{f(x-0)+f(x+0)}{2}. S(x)=2f(x−0)+f(x+0)​.

  • 若函数只在一个周期内给出，则必须先作周期延拓，再判断连接点处的连续性。
