题目描述

Pak Chanek 正在参加一场 lemper 烹饪比赛。在比赛中，Pak Chanek 需要用 NNN 个依次排列的炉灶（从第 111 个到第 NNN 个）来烹饪 lemper。最初，第 iii 个炉灶的温度为 AiA_iAi​ 度。一个炉灶的温度可以为负数。

Pak Chanek 意识到，为了让他的 lemper 能够被烹饪好，他需要保证每个炉灶的温度都不为负。为此，Pak Chanek 可以进行零次或多次操作。每次操作时，Pak Chanek 可以选择一个满足 2≤i≤N−12 \leq i \leq N-12≤i≤N−1 的炉灶 iii，然后：

  1. 将第 i−1i-1i−1 个炉灶的温度变为 Ai−1:=Ai−1+AiA_{i-1} := A_{i-1} + A_{i}Ai−1​:=Ai−1​+Ai​；
  2. 将第 i+1i+1i+1 个炉灶的温度变为 Ai+1:=Ai+1+AiA_{i+1} := A_{i+1} + A_{i}Ai+1​:=Ai+1​+Ai​；
  3. 将第 iii 个炉灶的温度变为 Ai:=−AiA_i := -A_iAi​:=−Ai​。

Pak Chanek 想知道，他最少需要进行多少次操作，才能使所有炉灶的温度都不为负。如果无法做到，请输出 −1-1−1。

输入格式

第一行包含一个整数 NNN（1≤N≤1051 \le N \le 10^51≤N≤105），表示炉灶的数量。

第二行包含 NNN 个整数 A1,A2,…,ANA_1, A_2, \ldots, A_NA1​,A2​,…,AN​（−109≤Ai≤109-10^9 \leq A_i \leq 10^9−109≤Ai​≤109），表示每个炉灶的初始温度。

输出格式

输出一个整数，表示使所有炉灶温度都不为负所需的最小操作次数。如果无法做到，输出 −1-1−1。
