Codeforces Round 254 (Div.1) C 线段树+珂朵莉树

发表于2026-06-27|更新于2026-06-27|算法
|浏览量:

题目描述

DZY 喜欢颜色，也喜欢绘画。

在一个五彩斑斓的日子里，DZY 得到了一条彩色丝带，这条丝带共有 nnn 个单位（编号从 111 到 nnn，从左到右）。第 iii 个单位初始颜色为 iii。虽然已经足够多彩，但我们仍然认为每个单位的“色彩度”一开始都是 000。

DZY 热爱绘画，如你所知。他拿起颜色为 xxx 的画笔，在丝带上画了条线。这种情况下，一些连续的单位会被涂色。假设某一时刻，第 iii 个单位的颜色为 yyy。当它被该画笔染色之后，颜色将变成 xxx，该单位的色彩度会增加 ∣x−y∣|x-y|∣x−y∣。

DZY 计划执行 mmm 次操作，每次操作有以下两种可能：

  1. 将编号在 lll 到 rrr 之间的所有单位（包含端点）统统用颜色 xxx 染色。
  2. 询问编号在 lll 到 rrr 之间的所有单位（包含端点）的色彩度之和。

你能帮 DZY 完成这一系列操作吗？

输入格式

第一行包含两个用空格分隔的整数 n,mn,mn,m（1≤n,m≤1051\leq n,m\leq 10^{5}1≤n,m≤105）。

接下来的 mmm 行，每行以一个整数 typetypetype（1≤type≤21\leq type\leq 21≤type≤2）开头，表示本次操作的类型。

如果 type=1type=1type=1，则本行还将包含三个整数 l,r,xl,r,xl,r,x（1≤l≤r≤n; 1≤x≤1081\leq l\leq r\leq n; 1\leq x\leq 10^{8}1≤l≤r≤n; 1≤x≤108），表示一次 1 操作。

如果 type=2type=2type=2，则本行还将包含两个整数 l,rl,rl,r（1≤l≤r≤n1\leq l\leq r\leq n1≤l≤r≤n），表示一次 2 操作。

输出格式

对于每次 2 操作，输出一行答案——所询问区间内的色彩度之和。

思路挺简单的，线段树维护区间和，珂朵莉树维护区间染色。
