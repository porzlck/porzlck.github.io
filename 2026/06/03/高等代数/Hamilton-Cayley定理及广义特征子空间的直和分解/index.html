Hamilton-Cayley 定理

设 AAA 是数域 PPP 上的 nnn 阶矩阵，记 AAA 的特征多项式为

f(λ)=∣λE−A∣.f(\lambda)=|\lambda E-A|. f(λ)=∣λE−A∣.

若

f(λ)=λn+a1λn−1+a2λn−2+⋯+an−1λ+an,f(\lambda) = \lambda^n+a_1\lambda^{n-1}+a_2\lambda^{n-2} +\cdots+a_{n-1}\lambda+a_n, f(λ)=λn+a1​λn−1+a2​λn−2+⋯+an−1​λ+an​,

则

f(A)=An+a1An−1+a2An−2+⋯+an−1A+anE=0.f(A) = A^n+a_1A^{n-1}+a_2A^{n-2} +\cdots+a_{n-1}A+a_nE = 0. f(A)=An+a1​An−1+a2​An−2+⋯+an−1​A+an​E=0.

这就是 Hamilton-Cayley 定理，也叫 Cayley-Hamilton 定理。

证明

设矩阵 λE−A\lambda E-AλE−A 的伴随矩阵为 B(λ)B(\lambda)B(λ)。

由伴随矩阵的性质可知，

B(λ)(λE−A)=∣λE−A∣E.B(\lambda)(\lambda E-A) = |\lambda E-A|E. B(λ)(λE−A)=∣λE−A∣E.

因为

f(λ)=∣λE−A∣,f(\lambda)=|\lambda E-A|, f(λ)=∣λE−A∣,

所以

B(λ)(λE−A)=f(λ)E.B(\lambda)(\lambda E-A) = f(\lambda)E. B(λ)(λE−A)=f(λ)E.

由于 B(λ)B(\lambda)B(λ) 是 λE−A\lambda E-AλE−A 的伴随矩阵，而 λE−A\lambda E-AλE−A 中每个元素关于 λ\lambdaλ 的次数不超过 111，所以 B(λ)B(\lambda)B(λ) 中每个元素关于 λ\lambdaλ 的次数不超过 n−1n-1n−1。

因此，B(λ)B(\lambda)B(λ) 可以写成

B(λ)=λn−1B0+λn−2B1+⋯+λBn−2+Bn−1,B(\lambda) = \lambda^{n-1}B_0 + \lambda^{n-2}B_1 + \cdots + \lambda B_{n-2} + B_{n-1}, B(λ)=λn−1B0​+λn−2B1​+⋯+λBn−2​+Bn−1​,

其中

B0,B1,…,Bn−1B_0,B_1,\ldots,B_{n-1} B0​,B1​,…,Bn−1​

都是 nnn 阶常数矩阵。

由

B(λ)(λE−A)=f(λ)EB(\lambda)(\lambda E-A) = f(\lambda)E B(λ)(λE−A)=f(λ)E

可得

(λn−1B0+λn−2B1+⋯+λBn−2+Bn−1)(λE−A)=f(λ)E.\left( \lambda^{n-1}B_0 + \lambda^{n-2}B_1 + \cdots + \lambda B_{n-2} + B_{n-1} \right) (\lambda E-A) = f(\lambda)E. (λn−1B0​+λn−2B1​+⋯+λBn−2​+Bn−1​)(λE−A)=f(λ)E.

左边展开：

B(λ)(λE−A)=λnB0+λn−1(B1−B0A)+λn−2(B2−B1A)+⋯+λ(Bn−1−Bn−2A)−Bn−1A.B(\lambda)(\lambda E-A) = \lambda^nB_0 + \lambda^{n-1}(B_1-B_0A) + \lambda^{n-2}(B_2-B_1A) + \cdots + \lambda(B_{n-1}-B_{n-2}A) - B_{n-1}A. B(λ)(λE−A)=λnB0​+λn−1(B1​−B0​A)+λn−2(B2​−B1​A)+⋯+λ(Bn−1​−Bn−2​A)−Bn−1​A.

右边为

f(λ)E=λnE+a1λn−1E+a2λn−2E+⋯+an−1λE+anE.f(\lambda)E = \lambda^nE + a_1\lambda^{n-1}E + a_2\lambda^{n-2}E + \cdots + a_{n-1}\lambda E + a_nE. f(λ)E=λnE+a1​λn−1E+a2​λn−2E+⋯+an−1​λE+an​E.

比较两边关于 λ\lambdaλ 的同次幂系数，得到

B0=E,B_0=E, B0​=E,

B1−B0A=a1E,B_1-B_0A=a_1E, B1​−B0​A=a1​E,

B2−B1A=a2E,B_2-B_1A=a_2E, B2​−B1​A=a2​E,

⋯\cdots ⋯

Bn−1−Bn−2A=an−1E,B_{n-1}-B_{n-2}A=a_{n-1}E, Bn−1​−Bn−2​A=an−1​E,

−Bn−1A=anE.-B_{n-1}A=a_nE. −Bn−1​A=an​E.

将上面这些等式依次右乘

An,An−1,An−2,…,A,E.A^n,A^{n-1},A^{n-2},\ldots,A,E. An,An−1,An−2,…,A,E.

得到

B0An=An,B_0A^n=A^n, B0​An=An,

B1An−1−B0An=a1An−1,B_1A^{n-1}-B_0A^n=a_1A^{n-1}, B1​An−1−B0​An=a1​An−1,

B2An−2−B1An−1=a2An−2,B_2A^{n-2}-B_1A^{n-1}=a_2A^{n-2}, B2​An−2−B1​An−1=a2​An−2,

⋯\cdots ⋯

Bn−1A−Bn−2A2=an−1A,B_{n-1}A-B_{n-2}A^2=a_{n-1}A, Bn−1​A−Bn−2​A2=an−1​A,

−Bn−1A=anE.-B_{n-1}A=a_nE. −Bn−1​A=an​E.

将以上所有等式左右两边分别相加。

左边发生逐项抵消：

B0An+(B1An−1−B0An)+(B2An−2−B1An−1)+⋯+(Bn−1A−Bn−2A2)−Bn−1A=0.B_0A^n + (B_1A^{n-1}-B_0A^n) + (B_2A^{n-2}-B_1A^{n-1}) + \cdots + (B_{n-1}A-B_{n-2}A^2) - B_{n-1}A = 0. B0​An+(B1​An−1−B0​An)+(B2​An−2−B1​An−1)+⋯+(Bn−1​A−Bn−2​A2)−Bn−1​A=0.

右边相加为

An+a1An−1+a2An−2+⋯+an−1A+anE.A^n + a_1A^{n-1} + a_2A^{n-2} + \cdots + a_{n-1}A + a_nE. An+a1​An−1+a2​An−2+⋯+an−1​A+an​E.

因此

An+a1An−1+a2An−2+⋯+an−1A+anE=0.A^n + a_1A^{n-1} + a_2A^{n-2} + \cdots + a_{n-1}A + a_nE = 0. An+a1​An−1+a2​An−2+⋯+an−1​A+an​E=0.

也就是

f(A)=0.f(A)=0. f(A)=0.

所以 Hamilton-Cayley 定理得证。

如果把特征多项式写成

f(λ)=∣λE−A∣,f(\lambda)=|\lambda E-A|, f(λ)=∣λE−A∣,

那么它通常具有形式

f(λ)=λn−(a11+a22+⋯+ann)λn−1+⋯+(−1)n∣A∣.f(\lambda) = \lambda^n -(a_{11}+a_{22}+\cdots+a_{nn})\lambda^{n-1} +\cdots +(-1)^n|A|. f(λ)=λn−(a11​+a22​+⋯+ann​)λn−1+⋯+(−1)n∣A∣.

因此代入矩阵 AAA 后得到

f(A)=An−(a11+a22+⋯+ann)An−1+⋯+(−1)n∣A∣E=0.f(A) = A^n -(a_{11}+a_{22}+\cdots+a_{nn})A^{n-1} +\cdots +(-1)^n|A|E = 0. f(A)=An−(a11​+a22​+⋯+ann​)An−1+⋯+(−1)n∣A∣E=0.

这正是 Hamilton-Cayley 定理的结论。

定理

设线性变换 AAA 的特征多项式为

f(λ)=(λ−λ1)r1(λ−λ2)r2⋯(λ−λs)rs.f(\lambda) = (\lambda-\lambda_1)^{r_1} (\lambda-\lambda_2)^{r_2} \cdots (\lambda-\lambda_s)^{r_s}. f(λ)=(λ−λ1​)r1​(λ−λ2​)r2​⋯(λ−λs​)rs​.

其中 λ1,λ2,…,λs\lambda_1,\lambda_2,\ldots,\lambda_sλ1​,λ2​,…,λs​ 是互不相同的特征值。

令

Vi=Ker⁡((A−λiE)ri),i=1,2,…,s.V_i = \operatorname{Ker}\left((A-\lambda_iE)^{r_i}\right), \qquad i=1,2,\ldots,s. Vi​=Ker((A−λi​E)ri​),i=1,2,…,s.

也就是说，

Vi={ξ∈V∣(A−λiE)riξ=0}.V_i = \left\{ \xi\in V \mid (A-\lambda_iE)^{r_i}\xi=0 \right\}. Vi​={ξ∈V∣(A−λi​E)ri​ξ=0}.

则

V=V1⊕V2⊕⋯⊕Vs.V = V_1\oplus V_2\oplus\cdots\oplus V_s. V=V1​⊕V2​⊕⋯⊕Vs​.

证明

设

fi(λ)=f(λ)(λ−λi)ri,i=1,2,…,s.f_i(\lambda) = \frac{f(\lambda)}{(\lambda-\lambda_i)^{r_i}}, \qquad i=1,2,\ldots,s. fi​(λ)=(λ−λi​)ri​f(λ)​,i=1,2,…,s.

于是

fi(λ)(λ−λi)ri=f(λ).f_i(\lambda)(\lambda-\lambda_i)^{r_i} = f(\lambda). fi​(λ)(λ−λi​)ri​=f(λ).

把 λ\lambdaλ 换成 AAA，得

fi(A)(A−λiE)ri=f(A).f_i(A)(A-\lambda_iE)^{r_i} = f(A). fi​(A)(A−λi​E)ri​=f(A).

由 Hamilton-Cayley 定理，

f(A)=0.f(A)=0. f(A)=0.

所以

fi(A)(A−λiE)ri=0.f_i(A)(A-\lambda_iE)^{r_i}=0. fi​(A)(A−λi​E)ri​=0.

因为 fi(A)f_i(A)fi​(A) 和 (A−λiE)ri(A-\lambda_iE)^{r_i}(A−λi​E)ri​ 都是 AAA 的多项式，所以可以交换顺序：

(A−λiE)rifi(A)=0.(A-\lambda_iE)^{r_i}f_i(A)=0. (A−λi​E)ri​fi​(A)=0.

于是对任意 α∈V\alpha\in Vα∈V，有

(A−λiE)rifi(A)α=0.(A-\lambda_iE)^{r_i}f_i(A)\alpha=0. (A−λi​E)ri​fi​(A)α=0.

所以

fi(A)α∈Ker⁡((A−λiE)ri)=Vi.f_i(A)\alpha \in \operatorname{Ker}\left((A-\lambda_iE)^{r_i}\right) = V_i. fi​(A)α∈Ker((A−λi​E)ri​)=Vi​.

因此

fi(A)V⊆Vi.f_i(A)V\subseteq V_i. fi​(A)V⊆Vi​.

因为

f1(λ),f2(λ),…,fs(λ)f_1(\lambda),f_2(\lambda),\ldots,f_s(\lambda) f1​(λ),f2​(λ),…,fs​(λ)

没有公共因式，所以存在多项式

u1(λ),u2(λ),…,us(λ),u_1(\lambda),u_2(\lambda),\ldots,u_s(\lambda), u1​(λ),u2​(λ),…,us​(λ),

使得

f1(λ)u1(λ)+f2(λ)u2(λ)+⋯+fs(λ)us(λ)=1.f_1(\lambda)u_1(\lambda) + f_2(\lambda)u_2(\lambda) + \cdots + f_s(\lambda)u_s(\lambda) = 1. f1​(λ)u1​(λ)+f2​(λ)u2​(λ)+⋯+fs​(λ)us​(λ)=1.

把 λ\lambdaλ 换成 AAA，得

f1(A)u1(A)+f2(A)u2(A)+⋯+fs(A)us(A)=E.f_1(A)u_1(A) + f_2(A)u_2(A) + \cdots + f_s(A)u_s(A) = E. f1​(A)u1​(A)+f2​(A)u2​(A)+⋯+fs​(A)us​(A)=E.

两边同时作用于任意向量 α∈V\alpha\in Vα∈V，得

f1(A)u1(A)α+f2(A)u2(A)α+⋯+fs(A)us(A)α=α.f_1(A)u_1(A)\alpha + f_2(A)u_2(A)\alpha + \cdots + f_s(A)u_s(A)\alpha = \alpha. f1​(A)u1​(A)α+f2​(A)u2​(A)α+⋯+fs​(A)us​(A)α=α.

即

α=f1(A)u1(A)α+f2(A)u2(A)α+⋯+fs(A)us(A)α.\alpha = f_1(A)u_1(A)\alpha + f_2(A)u_2(A)\alpha + \cdots + f_s(A)u_s(A)\alpha. α=f1​(A)u1​(A)α+f2​(A)u2​(A)α+⋯+fs​(A)us​(A)α.

因为

ui(A)α∈V,u_i(A)\alpha\in V, ui​(A)α∈V,

所以

fi(A)ui(A)α∈fi(A)V.f_i(A)u_i(A)\alpha\in f_i(A)V. fi​(A)ui​(A)α∈fi​(A)V.

又因为

fi(A)V⊆Vi,f_i(A)V\subseteq V_i, fi​(A)V⊆Vi​,

所以

fi(A)ui(A)α∈Vi.f_i(A)u_i(A)\alpha\in V_i. fi​(A)ui​(A)α∈Vi​.

因此任意 α∈V\alpha\in Vα∈V 都可以写成

α=α1+α2+⋯+αs,αi∈Vi.\alpha=\alpha_1+\alpha_2+\cdots+\alpha_s, \qquad \alpha_i\in V_i. α=α1​+α2​+⋯+αs​,αi​∈Vi​.

所以

V⊆V1+V2+⋯+Vs.V\subseteq V_1+V_2+\cdots+V_s. V⊆V1​+V2​+⋯+Vs​.

另一方面，因为每个 ViV_iVi​ 都是 VVV 的子空间，所以

V1+V2+⋯+Vs⊆V.V_1+V_2+\cdots+V_s\subseteq V. V1​+V2​+⋯+Vs​⊆V.

因此

V=V1+V2+⋯+Vs.V=V_1+V_2+\cdots+V_s. V=V1​+V2​+⋯+Vs​.

设

β1+β2+⋯+βs=0,\beta_1+\beta_2+\cdots+\beta_s=0, β1​+β2​+⋯+βs​=0,

其中

βi∈Vi,i=1,2,…,s.\beta_i\in V_i, \qquad i=1,2,\ldots,s. βi​∈Vi​,i=1,2,…,s.

因为

Vi=Ker⁡((A−λiE)ri),V_i = \operatorname{Ker}\left((A-\lambda_iE)^{r_i}\right), Vi​=Ker((A−λi​E)ri​),

所以

(A−λiE)riβi=0.(A-\lambda_iE)^{r_i}\beta_i=0. (A−λi​E)ri​βi​=0.

下面证明每个 βi\beta_iβi​ 都等于零。

固定一个 iii。

当 j≠ij\ne ij​=i 时，fi(λ)f_i(\lambda)fi​(λ) 中含有因子

(λ−λj)rj.(\lambda-\lambda_j)^{r_j}. (λ−λj​)rj​.

所以可以写成

fi(λ)=hij(λ)(λ−λj)rj.f_i(\lambda) = h_{ij}(\lambda)(\lambda-\lambda_j)^{r_j}. fi​(λ)=hij​(λ)(λ−λj​)rj​.

把 λ\lambdaλ 换成 AAA，得

fi(A)=hij(A)(A−λjE)rj.f_i(A) = h_{ij}(A)(A-\lambda_jE)^{r_j}. fi​(A)=hij​(A)(A−λj​E)rj​.

由于 βj∈Vj\beta_j\in V_jβj​∈Vj​，所以

(A−λjE)rjβj=0.(A-\lambda_jE)^{r_j}\beta_j=0. (A−λj​E)rj​βj​=0.

于是当 j≠ij\ne ij​=i 时，

fi(A)βj=hij(A)(A−λjE)rjβj=0.f_i(A)\beta_j = h_{ij}(A)(A-\lambda_jE)^{r_j}\beta_j = 0. fi​(A)βj​=hij​(A)(A−λj​E)rj​βj​=0.

现在对

β1+β2+⋯+βs=0\beta_1+\beta_2+\cdots+\beta_s=0 β1​+β2​+⋯+βs​=0

两边作用 fi(A)f_i(A)fi​(A)，得到

fi(A)β1+fi(A)β2+⋯+fi(A)βs=0.f_i(A)\beta_1 + f_i(A)\beta_2 + \cdots + f_i(A)\beta_s = 0. fi​(A)β1​+fi​(A)β2​+⋯+fi​(A)βs​=0.

由于 j≠ij\ne ij​=i 时 fi(A)βj=0f_i(A)\beta_j=0fi​(A)βj​=0，所以上式只剩下

fi(A)βi=0.f_i(A)\beta_i=0. fi​(A)βi​=0.

另一方面，由于 βi∈Vi\beta_i\in V_iβi​∈Vi​，所以

(A−λiE)riβi=0.(A-\lambda_iE)^{r_i}\beta_i=0. (A−λi​E)ri​βi​=0.

又因为

fi(λ)f_i(\lambda) fi​(λ)

和

(λ−λi)ri(\lambda-\lambda_i)^{r_i} (λ−λi​)ri​

互素，所以存在多项式 u(λ)u(\lambda)u(λ) 和 v(λ)v(\lambda)v(λ)，使得

u(λ)fi(λ)+v(λ)(λ−λi)ri=1.u(\lambda)f_i(\lambda) + v(\lambda)(\lambda-\lambda_i)^{r_i} = 1. u(λ)fi​(λ)+v(λ)(λ−λi​)ri​=1.

把 λ\lambdaλ 换成 AAA，得

u(A)fi(A)+v(A)(A−λiE)ri=E.u(A)f_i(A) + v(A)(A-\lambda_iE)^{r_i} = E. u(A)fi​(A)+v(A)(A−λi​E)ri​=E.

两边作用于 βi\beta_iβi​，得

u(A)fi(A)βi+v(A)(A−λiE)riβi=βi.u(A)f_i(A)\beta_i + v(A)(A-\lambda_iE)^{r_i}\beta_i = \beta_i. u(A)fi​(A)βi​+v(A)(A−λi​E)ri​βi​=βi​.

但

fi(A)βi=0f_i(A)\beta_i=0 fi​(A)βi​=0

所以

βi=0.\beta_i=0. βi​=0.

因为 iii 是任意的，所以

β1=β2=⋯=βs=0.\beta_1=\beta_2=\cdots=\beta_s=0. β1​=β2​=⋯=βs​=0.

因此这个和是直和：

V=V1⊕V2⊕⋯⊕Vs.V=V_1\oplus V_2\oplus\cdots\oplus V_s. V=V1​⊕V2​⊕⋯⊕Vs​.

定理得证。

总结

这个定理说明：

V=Ker⁡((A−λ1E)r1)⊕Ker⁡((A−λ2E)r2)⊕⋯⊕Ker⁡((A−λsE)rs).V = \operatorname{Ker}\left((A-\lambda_1E)^{r_1}\right) \oplus \operatorname{Ker}\left((A-\lambda_2E)^{r_2}\right) \oplus \cdots \oplus \operatorname{Ker}\left((A-\lambda_sE)^{r_s}\right). V=Ker((A−λ1​E)r1​)⊕Ker((A−λ2​E)r2​)⊕⋯⊕Ker((A−λs​E)rs​).

也就是说，整个空间 VVV 可以按照不同特征值拆成若干个互不重叠的广义特征子空间。
